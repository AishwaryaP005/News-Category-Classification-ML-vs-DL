# News Category Classification using Machine Learning and Deep Learning

## Overview
This project focuses on classifying news articles into predefined categories using both Machine Learning and Deep Learning techniques. The objective is to compare the effectiveness of traditional ML models against modern DL architectures for text classification tasks.

## Features
- Text preprocessing and cleaning
- TF-IDF feature extraction
- Multi-class news classification
- Comparative evaluation of ML and DL models
- Performance analysis using accuracy and classification metrics

## Models Implemented

### Machine Learning Models
- Support Vector Machine (SVM)
- Logistic Regression
- Naive Bayes
- Random Forest

### Deep Learning Models
- Artificial Neural Network (ANN)
- LSTM
- GRU

## Results

| Model | Accuracy |
|---------|----------|
| SVM | 80.0% |
| Logistic Regression | 79.6% |
| ANN | 79.0% |
| Naive Bayes | 77.6% |
| LSTM | 76.0% |
| GRU | 72.0% |
| Random Forest | 71.0% |

## Technologies Used
- Python
- Scikit-learn
- TensorFlow / Keras
- Pandas
- NumPy
- Matplotlib

## Conclusion
Traditional machine learning approaches outperformed deep learning models for this dataset, highlighting the effectiveness of TF-IDF-based representations for short-text classification tasks.
